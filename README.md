# CodeAlpha_Chatbot
A Python Tkinter-based chatbot with iMessage-style UI and random replies.

Project Overview

This project is a Python Tkinter-based chatbot designed to simulate a conversation with a user. It responds to predefined commands and provides random replies. The chatbot features an attractive iMessage-style interface and includes a send icon button for a modern UI.

Features

Random Replies: Bot replies are selected randomly from predefined lists using a dictionary.

iMessage-style Chat Bubbles: Blue bubble for user, grey bubble for bot.

Send Icon Button: Paper plane icon used for sending messages.

Smooth Scrolling: Chat window automatically scrolls to show the latest messages.

Keyboard Support: Press Enter to send messages.

Light/Dark Mode (Optional): Can be added for theme customization.

Technology Used

Python 3.x

Tkinter – for GUI

Random – for generating random replies
